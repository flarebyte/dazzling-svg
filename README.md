> Experimental library for processing svg images


## Install

```sh
$ npm install --save dazzling-svg
```


## Usage

```js
var dazzling-svg = require('dazzling-svg');

dazzling-svg('Rainbow');
```

```sh
$ npm install --global dazzling-svg
$ dazzling-svg --help
```

```sh
# creates a browser.js
$ npm run browser
```
